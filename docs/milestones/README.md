## Frontmatter conventions

- `milestone`: stable slug for filtering/grouping issue docs
- `milestone_doc`: wikilink to the source milestone document for scope and exit criteria
- `parent_issue`: parent issue number when this issue is a child of a broader issue
- `parent_issue_doc`: wikilink to the parent issue doc when applicable
- `child_issues`: child issue numbers for umbrella issues when they exist
- `child_issue_docs`: wikilinks to child issue docs for umbrella issues and draft child docs
- milestone notes intentionally do not carry issue lists in frontmatter; use Bases/ad-hoc queries instead
